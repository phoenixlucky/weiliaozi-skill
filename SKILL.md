# 尉缭子分析法 Skill

## 核心原则

按顺序想，不跳步：

> 先看结构，再看约束，再算利弊，最后定顺序与对抗策略。

## 五栏分析框架

### 1. 本质

先看问题的底层结构，不被表象带偏。

重点：真实驱动、核心变量、哪些是表面现象

### 2. 条件

再看现在有没有做这件事的基础。

重点：自身条件、外部条件、硬约束（无法靠意志突破的限制）

### 3. 得失

再算这件事值不值得做。

重点：短期vs长期收益、显性vs隐性成本、最坏情况能不能承受

### 4. 先后

再定顺序、节奏和路径。

重点：先解决瓶颈、分阶段推进、不一步到位

### 5. 对手

最后看博弈，对方不会静止不动。

重点：对方最可能的2-3种反应、你的动作会触发什么应对

## 工作顺序（不可跳步）

1. 定义问题（一句话）
2. 识别底层结构
3. 检查条件和硬约束
4. 计算收益、成本、风险
5. 安排顺序和路径
6. 模拟对手反应
7. 输出判断 + 建议动作

## 输出格式

每栏3-5个关键点，最后加：
- **判断一句**：整体结论
- **建议动作**：下一步1-3步

格式：Markdown 五栏表格 or 列表

## 快速判断模式

如果用户主要想知道"值不值得做"，优先填：
1. 条件（能启动吗）
2. 得失（赢了得到什么，输了亏多少）
3. 先后（第一步做什么）

## 风格禁忌

- 不先给建议，后补分析
- 不把表象当本质
- 不把愿望当条件
- 不只讲收益，不讲代价
- 不动作堆砌，无顺序
- 不默认对手不反应
